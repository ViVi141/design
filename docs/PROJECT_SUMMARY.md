## 🏗️ 技术架构

### 后端
```
FastAPI (Python 3.13)
├── SQLAlchemy - ORM
├── Pydantic - 数据验证
├── DeepSeek - AI服务
├── OR-Tools - 路线优化
└── 高德地图API - 地图服务
```

### 前端
```
Vue 3 + TypeScript
├── Element Plus - UI组件
├── Pinia - 状态管理
├── Vue Router - 路由
├── Vite - 构建工具
└── Axios - HTTP客户端
```
