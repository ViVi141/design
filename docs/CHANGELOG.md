# 更新日志

## [V3.0] - 2025-10-09

### 新增功能 ✨
- **一键AI生成**: 30秒自动生成完整行程
- **智能优化**: 一键优化路线顺序
- **撤销/重做**: 支持Ctrl+Z/Ctrl+Y，最多50步
- **实时统计**: 待安排、已安排、景点数统计
- **进度显示**: AI生成时显示进度条
- **快速开始面板**: 集中展示核心操作
- **操作工具栏**: 撤销、重做、清空按钮

### 改进 🔧
- 优化AI响应处理，支持多种格式
- 增强错误处理和用户提示
- 改进拖拽视觉反馈
- 优化页面布局为三栏设计

### 修复 🐛
- 修复axios拦截器导致的response.data问题
- 修复TypeScript类型错误
- 修复地图加载时的Window类型问题
- 删除旧版本文件（V1、V2）

### 文档 📚
- 新增README.md

## [V1.0] - 2025-10-01 至 10-05

### 核心功能 ✨
- **地图浏览**: 高德地图集成
- **景点搜索**: 基于高德API的搜索
- **AI对话**: DeepSeek集成
- **路线规划**: TSP算法优化
- **行程管理**: CRUD功能
- **数据库**: SQLite数据存储

### 技术栈 🛠️
- 后端: FastAPI + Python 3.13
- 前端: Vue 3 + TypeScript + Element Plus
- AI: DeepSeek API + LangChain
- 地图: 高德地图API
- 算法: Google OR-Tools
